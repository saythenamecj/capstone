Commands for Github(in terminal):

    link github Repository
        git remote add origin https://github.com/saythenamecj/capstone.git

    save changes
        git add .

    commit (add context for others)
        git commit -m "First Commit"

    push changes
        git push origin master

    make origin master default push location
        git push --set-upstream origin master


    For you:

    get latest update from repo
        git pull origin master
        git fetch origin master
        git reset --hard origin/master

    save changes
        git add .

    commit (add context for others)
        git commit -m "message"

    push (upload to repo)
        git push
